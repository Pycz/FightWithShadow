About

	Rarjpeg — картинка, склеенная с RAR-архивом (JPEG with embedded RAR-file). 
	Рарджепеги имеют наибольшее распространение, 
	хотя операция соединения графического файла и архива возможна также 
	для других графических форматов (PNG, GIF и т. д.) и архивов (ZIP, jar), 
	алсо аудио- и видеофайлов в формате .wma / .wav / .

Как запустить :

	Для запуска необходимо наличие на ЭВМ Java машины

	Лучше всего устанавливать JRE (Java Runtime Environment) -
	она меньше по объёму чем JDK (Java Development Kit)-- 
	это для программистов, для разработки программ на Java.

	Для домашнего компьютера подойдёт Java SE JRE-- 
	Standart Edition, для бизнес-пользователя Java EE -- Enterprise Edition.

	На данный момент актуальной является -- JRE 6 Update 12.

	После установки, просто двойной клик на иконке 
	или run.bat.
